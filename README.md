# README

## Setup

Ruby dependencies:

    bundle install

Javascript dependencies:

    yarn install

## Launch application

Back-end:

    rails s

Front-end:

    ./bin/webpack-dev-server

## Importing data

    rake data:import

## Linters and tests:

Rubocop

    bundle exec rubocop

Eslint

    yarn run eslint

Stylelint

    yarn run stylelint

Unit tests

    rspec
